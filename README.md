# GentleKyle's Basic HTML/CSS/JavaScript Sandbox Webpage 

## Description

This project is a collection of basic HTML, CSS, and JavaScript elements, created as I followed the W3Schools HTML tutorial. The webpage serves as a learning exercise and a showcase of various web development concepts. This is part of the Tensure Apprenticeship.

Tutorial: https://www.w3schools.com/html/default.asp

## Features

- Basic HTML structure and elements
- CSS styling for layout and design
- JavaScript for interactivity

## Prerequisites

- A web browser like Chrome, Firefox, Safari, or Edge

## How to Use

1. Clone or download the repository to your local machine.
2. Locate/navigate to the project directory.
3. Open your web browser.
4. Open the "index.html" file from the project directory. You can do this by either:
    - Dragging and dropping the "index.html" file into the browser window.
    - Right-clicking the "index.html" file and selecting "Open with" and then choosing your web browser.
    - Copy and paste the "index.html" file path into your browser's address bar.
5. Explore and enjoy!
